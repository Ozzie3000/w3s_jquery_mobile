Attribute Link

Data-attribute Value Description
data-ajax true | false Used to specify whether to load pages via ajax. This will improve performance in user experience and transitions. If set to false the normal jQuery mobile page request will occur
data-direction reverse Used to do the reverse transition for animation on dialog or page
data-dom-cache true | false Used to specify whether or not to clear the jQuery DOM cache. If set to true you will manage the DOM yourself
data-prefetch true | false Used to specify if prefetch pages into the DOM is used to be available to the visiting user
data-rel back | dialog | external | popup Used to specify an option for how the link show behave.
  • Back -goes back 1 step in history
  • Dialog - opens a link as a dialog so it is not tracked in history (really same page)
  • External - used for going to another domain
  • Popup - everyone's favorite the popup window
data-transition fade | flip | flow | pop | slide | slidedown | slideface | slideup | turn | none Used to specify the transition from one page to another
