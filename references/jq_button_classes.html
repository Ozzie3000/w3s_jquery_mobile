Button classes

You may add these classes to anchor or button elements. But not input buttons


Class Description
ui-btn Must be added to anchor elements to style like buttons
ui-btn-inline Displays button element inline
ui-btn-icon-top Diplays the icon on top of the text
ui-btn-icon-right Diplays the icon to the right side of the text
ui-btn-icon-bottom Diplays the icon to the bottom
ui-btn-icon-left Diplays the icon to the left
ui-btn-icon-notext Remove the text obviously
