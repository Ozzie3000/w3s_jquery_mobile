Collapsible set

Data-attribute Value Description
data-collapsed-icon Varies Used to specify the icon for a collapsible set. Default is the plus symbol
data-content-theme letter (a-z) Used to specify the them color for a collapsible set. c-z custom
data-expanded-icon Used to specify the icon for a collapsible button when the content is expanded. Default is the minus symbol
data-iconpos left | right | top | bottom | notext Used to specify the icon position
