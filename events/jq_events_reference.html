Events reference

Note: you will need to use the on() method with the events below

Event Description
hashchange Enables bookmarkable #hash history
navigate A wrapper event used for hashchange and popstate
orientationchange Triggered once the user rotates the mobile device vertically and horizonatally
pagebeforechange Triggered two times durign the page change cycle. Once prior to any page loading or transitioning. Then after page loading is complete, but before the browser history has been modified by the navigation process.
pagebeforecreate This event is triggered before the page is initialized, but before enhancement has begun
pagebeforehide Triggered on the originating page, before transition animation starts
pagebeforeload Old potatoes. Use pagecontainerbeforeload
pagechange Triggered after the changePage() request has finished loading the page into the DOM and all page transition animations have completed
pagechangefailed Triggered when the changePage() request fails to load the page. Callbacks for this particular event will be passed a data object as the 2nd argument
pagecontainerbeforeload Triggered before any load request is made
pagecontainerload Triggered after the page is successfully loaded and inserted into the DOM
pagecontainerloadfailed Triggered if the page load request fails
pagecreate Triggered when the page has been created, but before enhancement is complete
pagehide Triggered on the originating page after the transition animation completes
pageinit Don't use it. Use pagecreate
pageload Don't use it. user pagecontainerload
pageloadfailed Don't use it. Use pagecontainerloadfailed
pageremove Triggered before the framework attempts to remove an external page from the DOM
pageshow Triggered on the destination page after the transition animation is completed
scrollstart Triggered when the user starts to scroll the page
scrollstop Triggered when the user stops scrolling
swipe Triggered when the user swipes over an element horizontally
swipeleft Triggered when swiping to the left by more than 30px
swiperight Triggered when swiping to the right by more than 30px
tap Triggered when the user taps on an element
taphold Triggered when the user taps an element and holds it for 1,000 milliseconds
throttledresize Enables bookmarkable #hash history
updatelayout Triggered by jQuery mobile components that dynamically show / hide content
vclick A click event handler that is virtualized
vmousecancel mousecancel handler that is virtualized
vmousedown mousedown event handler that is virtualized
vmousemove mousemove event handler that is virtualized
